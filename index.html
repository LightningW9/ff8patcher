Final Fantasy VIII Remaster (Steam) DLL Patcher

This tool is an easy way to apply the UV texture patch created by Maki
It is powered by Bemani Patcher
The UV patch is automatically applied if you are using the FF8 demaster patch available here
1) Choose file C:\Program Files (x86)\Steam\steamapps\common\FINAL FANTASY VIII Remastered\FFVIII_EFIGS.dll
2) Tick the box to apply the UV patch
3) Save the patched DLL and overwrite the file (make a backup before overwriting)
